# 数据卡：HuChenFeng Persona SFT Early

## 数据集简介

这是一个中文人格风格 SFT 数据集，来源于公开的 HuChenFeng 相关转写材料，并经过 QA 抽取、人格化改写和严格过滤。

数据集目标是让模型学习更直接、更口语化、更接近特定人物表达习惯的回答方式。

它不以事实校正、客观中立或通用安全助手为主要目标。

## 文件说明

- `data/v1_clean_full.jsonl`
  - 清洗后的完整数据。
- `data/v1_train.jsonl`
  - 训练集。
- `data/v1_val.jsonl`
  - 验证集。
- `data/v1_test.jsonl`
  - 测试集。
- `data/v1_dpo_pairs.jsonl`
  - 可选偏好优化数据。
- `data/v1_report.json`
  - 数据统计信息。

## 构建流程

1. 对原始转写进行切片。
2. 从切片中抽取候选 QA。
3. 将候选回答改写为更直接、更口语化的人格风格回答。
4. 使用规则过滤和风格分数过滤，生成最终 v1 数据。
5. 按日期切分训练集、验证集和测试集，减少同一天近重复内容造成的数据泄漏。

## 推荐用途

- LoRA SFT 实验
- 中文人格风格模型研究
- QA 数据清洗与后处理研究
- 基于 `v1_dpo_pairs.jsonl` 的偏好优化实验

## 不推荐用途

- 事实问答基准
- 医疗、法律、金融等高风险决策
- 通用安全助手训练
- 需要客观、中立、温和表达的场景

## 局限性

- 数据集优化的是说话风格，不保证所有观点事实正确。
- 部分回答可能带有强烈判断、口语化表达或尖锐立场。
- 用该数据训练的模型可能过拟合某些口头表达。
- 如果将模型用于真实产品，请额外进行安全评测、事实评测和风格边界控制。
