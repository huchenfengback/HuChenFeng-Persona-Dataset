# 数据卡：HuChenFeng Persona SFT Mid

## 数据集简介

这是一个中文人格风格 SFT 数据集的中期版本，来源于公开转写材料，并经过 QA 抽取、本人化改写和规则过滤。

该版本的目标是训练模型更稳定地学习特定人物在中期语料中的表达方式：直接、口语化、判断明确，并尽量避免摘要腔和第三方转述腔。

## 文件说明

- `data/mid_clean_full.jsonl`：完整清洗集。
- `data/mid_train.jsonl`：训练集。
- `data/mid_val.jsonl`：验证集。
- `data/mid_test.jsonl`：测试集。
- `data/mid_dpo_pairs.jsonl`：可选偏好优化数据。
- `data/mid_report.json`：数据统计信息。

## 构建流程

1. 对候选 QA 进行规则过滤。
2. 使用本人化重写，将整理腔、旁白腔、第三方总结腔改为更自然的直接回答。
3. 对重写结果进行风格门控，过滤过短、过长、重复、直播上下文残留或摘要腔明显的样本。
4. 按日期切分训练集和验证集，减少近重复样本造成的评估泄漏。
5. 额外构造偏好优化数据，其中 `chosen` 是本人化回答，`rejected` 是重写前回答。

## 推荐用途

- LoRA SFT 实验
- 中文人格风格模型研究
- 口语化 QA 数据构建研究
- 偏好优化实验

## 不推荐用途

- 事实问答基准
- 高风险决策场景
- 通用安全助手训练
- 要求客观、中立、温和表达的产品场景

## 局限性

- 数据优化目标是表达风格，不保证所有观点事实正确。
- 部分回答可能包含强烈判断或尖锐立场。
- 训练后的模型可能学习到特定人物的表达偏好，需要根据应用场景另行评估。
